# Phase 5A Prompt – MVP Gate

Evaluate the MVP against docs/product/mvp-gate.md.

Do not proceed unless:
- customer can pay
- one core problem is solved
- scope is minimal
- unit economics are positive
- first 10 customer path is clear
- buildable in 60 days
- risks are acceptable

If approved, document the decision.
If not approved, redefine MVP.
