# Phase 2 Prompt – Business Validation

For the selected opportunity, create:
1. Validation plan
2. Interview script
3. Pricing test
4. Landing page copy
5. Cold outreach test
6. Competitor analysis
7. Unit economics hypothesis
8. Break-even model
9. First 10 customers plan

Do not write product code.

If validation confidence is weak, recommend returning to Phase 1.
