# Security Checklist

## App Security

- [ ] Input validation
- [ ] Output safety
- [ ] Auth checks
- [ ] Ownership checks
- [ ] Rate limiting
- [ ] Secure headers
- [ ] CSRF considered
- [ ] SSRF considered if external URLs exist

## Data Security

- [ ] PII minimized
- [ ] Secrets not committed
- [ ] Backups considered
- [ ] Retention defined
- [ ] Deletion/export path defined

## Dependency Security

- [ ] Dependency audit
- [ ] Dependabot
- [ ] No unmaintained critical package

## Operational Security

- [ ] Safe logs
- [ ] Incident procedure
- [ ] Secret rotation process
