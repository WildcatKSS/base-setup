# Freshness Policy

## Required Metadata

The following documents should contain a Last Updated field:

- docs/project-status.md
- docs/context/current-state.md
- docs/context/next-session.md

Format:

Last Updated: YYYY-MM-DD

## Governance Review

docs/governance/governance-review.md should be reviewed periodically.

## Goal

Keep repository memory current and reduce governance drift.
