# Decision Framework

Every meaningful decision must be evaluated against these criteria.

## Decision Criteria

1. Customer value
2. Revenue impact
3. Product-market fit impact
4. Unit economics impact
5. Security impact
6. Operational impact
7. Maintenance cost
8. Complexity cost
9. Reversibility
10. Evidence quality

## Decision Output

For significant decisions, document:

- Decision
- Context
- Options considered
- Chosen option
- Why this is simpler than alternatives
- What was intentionally not built
- Risks
- Reversal plan
- ADR link if applicable

## Lean Preference

When two options are both acceptable, choose the one with:
- fewer dependencies
- fewer files
- fewer moving parts
- lower operational burden
- lower maintenance cost
- easier rollback
